# 📍 住所マッパー (Address Mapper)

ポスティングやチラシ配りのルート整理を劇的にラクにする、ブラウザ完結型のWebツールです。
住所リストが記載されたPDFを読み込み、町名ごとに自動でグループ化して Google Maps の経路案内ルートを自動生成します。

## ✨ 特徴

* **📄 PDFから住所を一発抽出**
  PDF特有の不要な記号（箇条書きマークなど）や空白を自動でクリーニングし、正確な住所データだけを抜き出します。
* **🏘️ 町名で自動グループ化**
  抽出した住所を「〇〇区〇〇町」などの単位で自動的にまとめ、件数が多い順にリストアップします。
* **🗺️ スマホ対応の Google Maps ルート生成**
  グループ化された住所から、Google Mapsアプリでそのまま開ける「経路案内（ルート）URL」を生成します。パソコンからスマホへURLを送れば、現地ですぐにナビを開始できます。
* **🔒 完全ローカル動作（プライバシー保護）**
  住所データの読み込みや解析はすべてお使いのブラウザ上で行われます。外部サーバーにデータが送信されることはないため、機密情報が含まれるリストでも安心して使えます。

## 🚀 使い方

1. このプロジェクトの `index.html` をお使いのWebブラウザ（Google Chrome, Microsoft Edge など）で開きます。
2. 画面中央のドロップゾーンに、住所リストが含まれた **PDFファイル** をドラッグ＆ドロップ（またはクリックして選択）します。
3. 自動で解析が始まり、画面下部に町名ごとのグループが表示されます。
4. **「経路を開く」** ボタンをクリックすると、対象の住所を巡回する Google Maps の経路案内が開きます。
   * 個別のピンだけを見たい場合は、各住所の横にある「📌 この住所」ボタンを押してください。

## ⚠️ 注意事項

* **Google Mapsの経由地制限について**
  Google Mapsの経路案内システムの仕様上、1回のルート検索で指定できる地点は **最大10件（出発地1 + 経由地8 + 目的地1）** までとなっています。1つのグループに11件以上の住所がある場合、本ツールでは自動的に「最初の10件」を抽出してルートを作成します。
* **「すべてのグループを開く」機能について**
  複数のルートを一気にブラウザの別タブで開く機能ですが、ブラウザの「ポップアップブロック」が有効になっていると最初の1つしか開かない場合があります。使用する際は、ブラウザのアドレスバー付近からポップアップを「許可」に設定してください。

## 🛠️ 使用技術

* **HTML / CSS / Vanilla JavaScript**: フレームワークなしの軽量構成です。
* **[PDF.js](https://mozilla.github.io/pdf.js/) (v3.11.174)**: PDFファイルのテキスト解析に使用しています（CDN経由で読み込み）。
* **Google Maps URLs API**: スマホアプリでも確実に経路として読み込める公式API形式のURLを生成しています。
